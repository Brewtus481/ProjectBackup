# Summary

Yet another toothpaste roller. The others I've seen seemed too flimsy, this is a printed thing I actually use every day.

# Print Settings

Printer Brand: Prusa
Printer: Prusa Clone
Rafts: Yes
Supports: Yes
Resolution: I used 0.15 with a 0.3mm nozzle
Infill: 20%

Notes: 
The spool should be printed on end with the handle down, with a brim, but no supports.

The cage is a bit tricky as it needs supports but you need to be able to get them out when it is done. I included my solution as an STL file.
