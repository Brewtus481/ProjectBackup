# Summary

This is a Cat fidget Spinner I designed. It is designed to be on the smaller side so it should fit in kids hands, but adults can play with it too!

This requires 608ZZ bearings.

The bearing covers also have a small lip to prevent the covers from accidentally touching the bearing and slowing it down. You can squeeze these covers firmly without deforming them and slowing the bearings.

***Use my Spinner Press to help insert the bearings. http://www.thingiverse.com/thing:2275122***
